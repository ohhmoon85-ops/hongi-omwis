# OMWIS 개발 현황 보고 (회장님 보고용)

**(주)홍지 주문·배송·창고·재고 통합 관리 시스템 — OMWIS**
보고일: 2026-07-01 · 작성: 개발팀

---

## 1. 한 줄 요약

> **PCB용 알루미늄 엔트리시트 D2C(직거래) 사업을, 주문 접수부터 재고·세금계산서까지 한 화면에서 처리하는 시스템을 구축하여 실서비스로 배포 완료했습니다.** 현재 실제 단가 입력과 거래처 계정 발급만 남은 상태입니다.

- **접속:** https://hongi-omwis.vercel.app (실서버 운영 중)
- **기반:** 클라우드(Vercel + Supabase), 실시간 DB, 5계층 권한 보안

---

## 2. 무엇을 할 수 있는가 (핵심 기능 — 모두 구현·배포 완료)

| 영역 | 기능 |
|---|---|
| **거래처 주문** | 거래처가 직접 온라인 주문 → 관리자 화면에 **실시간** 자동 표시·알림 |
| **주문 처리** | 승인·거절·출고 완료, 출고 시 **재고 자동 차감(FIFO)**, 반품 처리 |
| **세금계산서** | 부가세 자동계산, 전자세금계산서 발행(팝빌 연동 준비), 인쇄·PDF |
| **재고 관리** | 입고·Lot 관리, 안전재고 미달 경보, **소진 예측**, 수입신고필증 첨부 |
| **단가 체계** | 품목 기본단가 + **거래처별 협상가**(우선 적용), 가격 위·변조 방지(서버 검증) |
| **품목 관리** | 알루미늄 1050 H18 카탈로그(종류 × 사이즈 × 순도), 종류별 트리 뷰 |
| **경영 대시보드** | 오늘 주문·매출·미수금·재고경보 등 **실시간 KPI + 그래프** |
| **회장 모니터링** | 매출 추세·거래처 순위 차트, 환율·원자재 시세, **ACIS 구매신호** (Read-Only) |
| **ACIS 연동** | 알루미늄 수입 의사결정 시스템과 실시간 연동 (구매 타이밍 신호) |
| **자동화** | 데일리(18시)·주간(월 9시) 경영요약 자동 이메일, 카카오 알림톡 |

**권한 체계 (5계층):** 회장(전사 열람) · 슈퍼관리자(대표) · 운영관리자 · 거래처 — 각자 필요한 것만 보이도록 보안 격리.

---

## 3. 사업적 의미

- **인건비 절감:** 주문 접수·재고 대사·세금계산서를 자동화 → 담당자별 수작업 최소화
- **실시간 가시성:** 회장·대표가 언제든 전사 현황을 휴대폰으로 열람
- **수입 최적화:** ACIS 신호로 알루미늄 구매 타이밍 판단 → 원가 경쟁력
- **확장성:** 거래처·품목·직원 수가 늘어도 구조 변경 없이 수용

## 5. 정식 오픈(Go-Live) 전 남은 작업

1. **실제 단가표 입력** — 현재 생알미늄만 예시가, 전 품목 실단가 반영 필요
2. **외부 연동 키** — 카카오 알림톡 / 팝빌 세금계산서 / 이메일 발송 (사업자 계정 확보 시 즉시 연결)
3. **거래처 실계정 발급 + 주소 공유** — 준비된 발급 화면으로 즉시 가능
4. **법인통장·세금계산서 API 연동** — 수입 결제·정산 자동화(2단계 과제)

> 위 항목은 **개발이 아니라 운영 정보·외부 계정 확보**의 문제로, 자료만 확보되면 단기 반영 가능합니다.
