02 — 사업적 가치

왜 좋은가

  • 인건비 절감 주문 접수·재고 대사·세금계산서 발행을 자동화 → 수작업 최소화
  • 실시간 가시성 회장·대표가 언제든 휴대폰으로 전사 현황을 열람
  • 수입 원가 경쟁력 ACIS 수입신호로 알루미늄 구매 타이밍 판단 → 원가 최적화
  • 확장성 거래처·품목·직원이 늘어도 구조 변경 없이 수용
  • 보안 회장 / 대표 / 운영 / 거래처 — 5계층 권한으로 각자 필요한 것만
03 — 현재 상태

이미 실서버에서 동작합니다

  • 시스템 개발 · 클라우드 배포 완료 — 실서버 운영 중
  • 주문 → 실시간 접수 → 승인 → 세금계산서 → 출고 → 재고차감 전 흐름 검증
  • 실 품목 카탈로그 등록 — 알루미늄 1050 H18 (종류 × 사이즈 × 순도)
  • 모바일 · 주간/야간 테마 대응
  • 자동화 — 데일리(18시)·주간(월 9시) 경영요약 이메일, 카카오 알림톡
04 — 정식 오픈까지

남은 일 & 요청

  • 실제 단가표 입력 현재 생알미늄만 예시가 — 전 품목 실단가 반영
  • 외부 연동 키 확보 카카오 알림톡 · 팝빌 세금계산서 · 이메일 발송 계정
  • 거래처 실계정 발급 + 접속주소 공유 발급 화면 준비 완료 → 즉시 가능
  • (2단계) 법인통장 · 세금계산서 API 연동 수입 결제·정산 자동화
핵심: 남은 항목은 개발이 아니라 운영 정보·외부 계정 확보의 문제입니다. 자료만 확보되면 단기 반영 → 정식 오픈 가능합니다.
